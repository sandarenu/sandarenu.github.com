Caution with Java Autoboxing

In Java Boxing means when you convert a primitive type to a reference type. So when you have some thing like boolean myBool = new Boolean("true"); Boolean object is converted to primitive type boolean. So how this happens under the hood? Java compiler uses the method booleanValue() in Boolean class to get the primitive value. This is where you have to be careful; there is a possibility of NPE when the Boolean object is null. »
